C++ Command Line Hangman Game

Picks random film from a given list. Player wins if they guess the film's letters before they run out of incorrect guesses.
- Add or remove additional films as you please to expand/restrict the choices. - Handles letter guesses, full aswer guesses, case sensitivity and more.
- Hangman Style procedule drawing for loss of lifes.
- Console clears after each guess + more visual and QOL features.
